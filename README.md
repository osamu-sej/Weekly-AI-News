# Astral Option: Neon Wing

ブラウザだけで遊べる、横スクロール型のネオン・シューティングゲームです。3つのステージを進み、各ステージ終盤の大型ボスを撃破しながら高スコアを目指します。敵を倒すと出現する装備カプセルで、随伴機・スプレッドショット・ホーミングミサイル・レーザー・シールドを強化できます。

## 遊び方

1. `index.html` をブラウザで開きます。
2. `START MISSION` を押して出撃します。
3. 敵を倒してスコアとコンボを伸ばし、色分けされた装備カプセルを回収します。
4. 3面それぞれの終盤に出現するボスを撃破し、ミッションクリアを目指します。

## 操作

- 移動: `WASD` / `矢印キー` / マウス・タッチのドラッグ
- 通常ショット: 自動連射 / `Space`
- レーザー: `Shift` / 右クリック
- 低速移動: `Alt`

## 装備カプセル

- `O` / 黄色: 最大4機まで増える随伴オプション
- `S` / シアン: 斜め弾を追加するスプレッドキャノン
- `M` / ピンク: 自動追尾するホーミングポッド
- `L` / グリーン: レーザーゲージ回復
- Shield / 白: シールド回復

## 特徴

- 月光残骸帯、紫電アステロイド、ネオン要塞中枢の3面構成
- 各面の最後に大型ボスが登場し、専用HPバーと弾幕パターンで攻撃
- 最大4機のオプション、3段階スプレッド、3段階ホーミングで火力を段階的に増強
- プレイヤー機、随伴機、5種類の敵機、ボスのシルエットを描き分け
- シールド、ボスHP/コンボ、レーザーゲージ、Stage表示を統合したHUD
- Canvasで描画するパーティクル、スクリーンシェイク、グロウ、ステージ別背景
- キーボード、マウス、タッチ操作に対応したレスポンシブUI
