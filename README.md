# Astral Option: Neon Wing

ブラウザだけで遊べる、横スクロール型のネオン・シューティングゲームです。敵を倒すと出現するオプションカプセルを回収し、随伴機とレーザー火力を強化しながら高スコアを目指します。

## 遊び方

1. `index.html` をブラウザで開きます。
2. `START MISSION` を押して出撃します。
3. 敵を倒してスコアとコンボを伸ばし、黄色いオプションカプセルを回収します。

## 操作

- 移動: `WASD` / `矢印キー` / マウス・タッチのドラッグ
- 通常ショット: 自動連射 / `Space`
- レーザー: `Shift` / 右クリック
- 低速移動: `Alt`

## 特徴

- 最大4機のオプションがプレイヤーに追従し、火力を段階的に増強
- シールド、コンボ、レーザーゲージ、Wave表示を統合したHUD
- Canvasで描画するパーティクル、スクリーンシェイク、グロウ、星雲背景
- キーボード、マウス、タッチ操作に対応したレスポンシブUI
